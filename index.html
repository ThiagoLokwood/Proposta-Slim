Dúvidas Frequentes

Posso trocar os jogos no futuro?

Sim. No modelo de venda, novos módulos de jogos podem ser adquiridos sob demanda. No modelo híbrido, você tem direito a até 4 alterações mensais de layout (logos e artes), permitindo adaptar o totem para diferentes eventos dentro do mesmo mês.

Como funciona a estabilidade do sistema?

Os jogos rodam 100% localmente no Mini PC. Isso significa que não dependemos de internet no local do evento para o funcionamento do jogo, garantindo zero travamentos e performance máxima.

Vamos criar algo incrível juntos?

Esta proposta é flexível e pode ser ajustada conforme a escala do seu projeto ou as necessidades específicas de cada locação.
